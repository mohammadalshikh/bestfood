# BestFood - Java Spring Boot

## Minimum required

- Java installed (validated in this repo with Java 25)

## Run

1- From repo root:

```bash
chmod +x mvnw
./mvnw -DskipTests spring-boot:run
```

2- Open http://localhost:8080

Stop with `Ctrl + C`.
